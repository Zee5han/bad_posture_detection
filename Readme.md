# Bad Posture Detection
A background computer-vision system that detects posture and mouth breathing using MediaPipe and triggers a system-wide overlay after prolonged bad posture.

## Environment Notes
This project uses:
- Python 3.10
- NumPy 1.26.4
- MediaPipe 0.10.9

MediaPipe is currently incompatible with NumPy 2.x.
Please follow the installation instructions exactly.
